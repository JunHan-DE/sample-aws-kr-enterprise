# Analytics

AWS 데이터 분석 서비스를 활용한 데모 및 샘플 코드 모음입니다. Amazon Athena를 통한 서버리스 SQL 분석, AWS Glue 기반 ETL 파이프라인 구축, Amazon Kinesis의 실시간 스트리밍 처리, 그리고 Amazon Redshift 데이터 웨어하우스까지 데이터 수집에서 시각화까지의 전체 분석 파이프라인을 다룹니다. 대규모 데이터를 효율적으로 처리하고 인사이트를 도출하기 위한 아키텍처 패턴과 모범 사례를 제공합니다.

## 대표 서비스

- [Amazon Athena](https://aws.amazon.com/ko/athena/) - 표준 SQL로 Amazon S3 데이터를 분석하는 서버리스 대화형 쿼리 서비스
- [AWS Glue](https://aws.amazon.com/ko/glue/) - 데이터를 대규모로 검색, 준비 및 통합하는 서버리스 데이터 통합 서비스
- [Amazon Kinesis](https://aws.amazon.com/ko/kinesis/) - 실시간 비디오 및 데이터 스트림 수집, 처리, 분석 서비스
- [Amazon OpenSearch Service](https://aws.amazon.com/ko/opensearch-service/) - AI 기반 검색, 관찰성 및 벡터 데이터베이스를 위한 관리형 서비스
- [Amazon QuickSight](https://aws.amazon.com/ko/quicksight/) - AI 기반 비즈니스 인텔리전스(BI) 서비스
- [Amazon Redshift](https://aws.amazon.com/ko/redshift/) - 데이터 레이크하우스용 SQL 기반 클라우드 데이터 웨어하우스
- [Amazon EMR](https://aws.amazon.com/ko/emr/) - Apache Spark 등 빅 데이터 워크로드를 손쉽게 실행하는 관리형 플랫폼
