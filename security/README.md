# Security

IAM, Cognito, KMS 등 보안 및 자격 증명 서비스를 활용한 데모 및 샘플 코드를 제공합니다. AWS 보안 서비스는 자격 증명 관리, 데이터 암호화, 네트워크 보호, 위협 탐지, 규정 준수 등 클라우드 보안의 모든 계층을 포괄합니다. 제로 트러스트 아키텍처 구현, 인증/인가 시스템 구축, 암호화 키 관리, 웹 애플리케이션 방화벽 설정, 보안 이벤트 모니터링 등 엔터프라이즈 수준의 보안 요구 사항을 충족하는 데 활용됩니다. AWS는 공동 책임 모델을 기반으로 클라우드 인프라 보안을 제공하며, 고객은 이러한 서비스를 활용하여 클라우드 내 워크로드를 안전하게 보호할 수 있습니다.

## 대표 서비스

- [AWS IAM](https://aws.amazon.com/ko/iam/) - AWS 서비스 및 리소스에 대한 액세스와 자격 증명을 안전하게 관리하는 서비스
- [Amazon Cognito](https://aws.amazon.com/ko/cognito/) - 안전하고 확장 가능한 사용자 지정 가입 및 로그인 환경을 손쉽게 구현할 수 있는 서비스
- [AWS KMS](https://aws.amazon.com/ko/kms/) - 데이터를 암호화하거나 디지털 서명할 때 사용하는 키를 생성 및 제어하는 서비스
- [AWS WAF](https://aws.amazon.com/ko/waf/) - 일반적인 웹 익스플로잇으로부터 웹 애플리케이션을 보호하는 웹 애플리케이션 방화벽
- [Amazon GuardDuty](https://aws.amazon.com/ko/guardduty/) - 지능형 위협 탐지 기능으로 AWS 계정, 워크로드 및 데이터를 보호하는 서비스
- [AWS Security Hub](https://aws.amazon.com/ko/security-hub/) - 보안 문제의 우선순위를 정하고 대규모로 보안 상태를 관리할 수 있는 통합 보안 서비스
