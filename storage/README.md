# Storage

S3, EFS, FSx 등 다양한 스토리지 서비스를 활용한 데모 및 샘플 코드를 제공합니다. AWS 스토리지 서비스는 객체, 블록, 파일 등 다양한 스토리지 유형을 제공하여 데이터의 특성에 맞는 최적의 저장 방식을 선택할 수 있습니다. 정적 웹사이트 호스팅, 데이터 레이크 구축, AI/ML 학습 데이터 관리, 엔터프라이즈 파일 공유, 백업 및 복구 등 폭넓은 유스케이스에 활용됩니다. 높은 내구성과 가용성을 갖추고 있으며, 사용량에 따른 요금 모델로 비용 효율적인 데이터 관리가 가능합니다.

## 대표 서비스

- [Amazon S3](https://aws.amazon.com/ko/s3/) - 어디서나 원하는 양의 데이터를 저장하고 검색할 수 있는 객체 스토리지 서비스
- [Amazon S3 Vectors](https://aws.amazon.com/ko/s3/features/vectors/) - 대규모 벡터 저장 및 쿼리를 기본 지원하는 비용 최적화된 AI 지원 스토리지
- [Amazon EBS](https://aws.amazon.com/ko/ebs/) - 어떤 규모로도 사용하기 쉬운 고성능 블록 스토리지 서비스
- [Amazon EFS](https://aws.amazon.com/ko/efs/) - 서버리스 방식의 완전 탄력적인 공유 파일 스토리지 서비스
- [Amazon FSx](https://aws.amazon.com/ko/fsx/) - 다양한 기능의 고성능 파일 시스템을 손쉽게 시작, 실행 및 규모 조정할 수 있는 서비스
- [AWS Backup](https://aws.amazon.com/ko/backup/) - AWS 서비스 전반의 데이터 보호를 중앙에서 관리하고 자동화하는 서비스
